## 🌐 온라인 배포

### ✅ Vercel에 배포하기 (추천 - 5분)

#### 1단계: GitHub 준비
```bash
# 이 폴더를 GitHub에 업로드
git init
git add .
git commit -m "first commit"
git branch -M main
git remote add origin https://github.com/[your-username]/culture-skills-mapper.git
git push -u origin main
```

#### 2단계: Vercel 배포
1. https://vercel.com 방문
2. GitHub 계정으로 로그인
3. "New Project" 클릭
4. 저장소 선택 (culture-skills-mapper)
5. "Deploy" 클릭
6. 완료! ✨

**결과:** `https://culture-skills-mapper-[your-name].vercel.app` 형태의 URL 생성

## 💻 로컬 개발 환경 설정

### 준비물
- Node.js 16+ (https://nodejs.org 에서 다운로드)
- Git (https://git-scm.com 에서 다운로드)
- 텍스트 에디터 (VS Code 추천)

### 설치 (3단계)

#### 1단계: Node.js 설치 확인
```bash
node -v
npm -v
```

#### 2단계: 의존성 설치
```bash
npm install
```

#### 3단계: 개발 서버 실행
```bash
npm run dev
```

**결과:** 브라우저에서 `http://localhost:5173` 자동으로 열림

## 💾 데이터 저장하기

### 현재 상태
- 브라우저 메모리에만 저장됨
- 새로고침하면 데이터 사라짐

### 영구 저장 옵션

#### 옵션 1: 로컬 스토리지 (가장 간단)
```javascript
// src/App.jsx 에서 useEffect 추가
useEffect(() => {
  localStorage.setItem('behaviors', JSON.stringify(behaviors));
}, [behaviors]);

// 초기화 시 불러오기
useEffect(() => {
  const saved = localStorage.getItem('behaviors');
  if (saved) setBehaviors(JSON.parse(saved));
}, []);
```

#### 옵션 2: Google Sheets (연동)
```javascript
npm install axios

// Google Sheets API로 저장
```

#### 옵션 3: Firebase (구글 무료 DB)
```javascript
npm install firebase

// Firebase 설정 후 실시간 동기화
```

## ❓ FAQ

### Q1: 비용이 얼마나 드나?
**A:** Vercel/Netlify는 완전히 무료! (월간 대역폭 제한 없음)

### Q2: 데이터가 안전한가?
**A:** Vercel/Netlify는 매우 안전합니다. 엔터프라이즈급 보안.

### Q3: 몇 명이 동시에 사용할 수 있나?
**A:** 무제한! (데이터 저장은 별도 설정 필요)

### Q4: 인터넷 없이 사용 가능한가?
**A:** 현재는 불가능. 앞으로 오프라인 모드 추가 가능.

### Q5: 조직의 색상으로 커스터마이징 가능한가?
**A:** 네! `tailwind.config.js`에서 색상 변경 가능.

### Q6: 모바일에서도 사용 가능한가?
**A:** 네! 반응형 디자인으로 모든 기기에서 동작.

### Q7: 추가 기능을 원하면?
**A:** `src/App.jsx` 수정하거나 개발자에게 의뢰.

### Q8: GitHub 계정이 없으면?
**A:** GitHub 가입 (무료) → Vercel 배포
