### 옵션 2: 로컬에서 개발 (30분)

#### 요구사항
- Node.js 16+ (https://nodejs.org)
- Git (https://git-scm.com)
- VS Code 같은 에디터

#### 단계
```bash
# 1. Node.js 설치 확인
node -v
npm -v

# 2. 이 폴더에서 실행
npm install

# 3. 개발 서버 시작
npm run dev

# 4. http://localhost:5173 에서 확인
```

👉 자세한 가이드: `빠른_시작_가이드.md` 참고

## ❓ 자주 묻는 질문

### Q1: 정말 무료인가요?
**A:** 네! Vercel/Netlify 모두 무료입니다. 비용 0원입니다.

### Q2: 보안은 괜찮나요?
**A:** Vercel은 엔터프라이즈급 보안입니다. 큰 회사들이 사용합니다.

### Q3: 몇 명이 사용할 수 있나요?
**A:** 제한 없습니다. 단, 데이터 저장은 별도 설정 필요.

### Q4: 인터넷 끊기면?
**A:** 현재는 온라인 전용입니다. 오프라인 모드는 나중에 추가 가능.

### Q5: 모바일에서도 되나요?
**A:** 네! 모든 기기에서 완벽하게 작동합니다.

### Q6: 코드 수정이 어려울 것 같은데?
**A:** 처음 하신다면 가이드를 따르면 됩니다. 어려우면 개발자에게 의뢰.

### Q7: 깃허브 / Node.js가 뭔가요?
**A:** 걱정하지 마세요! 가이드에 따라 하면 됩니다.

### Q8: 배포 후에 수정할 수 있나요?
**A:** 네! 코드 수정 → GitHub에 푸시 → 자동으로 배포됨.
