# 조직문화 행동-스킬셋 평가 앱 설치 및 배포 가이드

## 방법 1️⃣: 가장 쉬운 방법 - 무료 웹 호스팅 (추천)

### 1. Vercel에 배포 (5분)

**필요한 것:**
- GitHub 계정 (없으면 가입)
- Vercel 계정 (GitHub으로 가입 가능)

**단계:**

1. **GitHub에 프로젝트 생성**
   ```bash
   # 로컬 폴더 생성
   mkdir culture-skills-mapper
   cd culture-skills-mapper
   
   # Git 초기화
   git init
   ```

2. **필수 파일 생성**
   - `package.json` 생성
   - `public/index.html` 생성
   - `src/App.jsx` 에 위의 React 코드 저장
   - `src/index.jsx` 생성

3. **GitHub에 푸시**
   ```bash
   git add .
   git commit -m "Initial commit"
   git branch -M main
   git remote add origin https://github.com/your-username/culture-skills-mapper.git
   git push -u origin main
   ```

4. **Vercel에서 배포**
   - https://vercel.com 방문
   - "New Project" 클릭
   - GitHub 저장소 선택
   - Deploy 클릭
   - 완료! (자동으로 URL 생성됨)

**비용:** 무료
**장점:** 가장 쉬움, 자동 SSL, 무료 도메인

## 완전한 프로젝트 설정 (Local 개발)

### 1단계: Node.js 설치
- https://nodejs.org/ 에서 LTS 버전 다운로드
- 설치 후 `node -v` 로 확인

### 2단계: React 프로젝트 생성

```bash
# Create React App 사용
npx create-react-app culture-skills-mapper
cd culture-skills-mapper

# 또는 Vite 사용 (더 빠름)
npm create vite@latest culture-skills-mapper -- --template react
cd culture-skills-mapper
npm install
```

### 3단계: 필요한 패키지 설치

```bash
npm install lucide-react
```

### 4단계: 코드 작성

`src/App.jsx` 에 위의 React 코드를 복사하고 저장합니다.

### 5단계: 실행

```bash
npm start
# 또는 (Vite의 경우)
npm run dev
```

자동으로 브라우저에 `http://localhost:3000` 열림
