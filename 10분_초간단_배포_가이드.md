## 🚀 3단계 배포

### **1단계: GitHub에 업로드 (2분)**

#### 방법 A: 웹으로 하기 (가장 쉬움)
1. https://github.com/new 방문
2. Repository name: `culture-skills-mapper` 입력
3. "Create repository" 클릭
4. "uploading an existing file" 클릭
5. 다운받은 폴더의 모든 파일 드래그&드롭
6. "Commit changes" 클릭

#### 방법 B: 명령어로 하기
```bash
# 1. 이 폴더에서
git init
git add .
git commit -m "first"
git branch -M main

# 2. 위에서 만든 저장소 주소를 복사해서 아래에 붙여넣기
git remote add origin https://github.com/[your-username]/culture-skills-mapper.git

# 3. 업로드
git push -u origin main
```

---

### **2단계: Vercel에 연결 (5분)**

1. **https://vercel.com 방문**
2. **GitHub으로 로그인** (또는 가입)
3. **"Add New"** 클릭 → **"Project"** 선택
4. **"Import Git Repository"** 클릭
5. **culture-skills-mapper 검색** 후 선택
6. **"Import"** 클릭
7. **"Deploy"** 클릭
8. **잠깐 기다리기...**

---

### **3단계: 완료! ✨ (3분)**

자동으로 배포되고, 다음과 같은 URL이 생성됩니다:
```
https://culture-skills-mapper-[your-name].vercel.app
```

이 URL을 브라우저에 입력하면 앱이 떠있습니다! 🎉

## 🆘 문제 해결

### Q: GitHub 계정이 없는데?
**A:** https://github.com/signup 에서 무료 가입 (1분)

### Q: Vercel이 뭔데?
**A:** 무료 웹 호스팅 서비스. 최고 수준의 속도와 보안 보장

### Q: 비용이 드나?
**A:** 완전히 무료! (대역폭 무제한)

### Q: 배포가 안 되면?
**A:** 
1. GitHub에 모든 파일이 있는지 확인
2. `package.json` 이 최상위 폴더에 있는지 확인
3. Vercel 대시보드의 "Deployments" 탭에서 에러 메시지 확인

### Q: 데이터를 저장하고 싶은데?
**A:** 현재는 브라우저 메모리에 저장됨. 필요시 나중에 Google Sheets나 Firebase 연동 가능
